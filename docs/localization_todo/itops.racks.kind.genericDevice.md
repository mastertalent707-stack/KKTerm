# itops.racks.kind.genericDevice

- **English value**: `Generic Device`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/RackItemDialog.tsx`
- **UI role**: `label`
- **User flow**: Shown as the merged Rack Device type in Rack Device pickers, previews, callouts, and exports.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: A nonspecific physical device installed in a Rack, replacing the previously separate Equipment and General types.
