# settings.schemeMatchOs

- **English value**: `Match OS`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/AppearanceSettings.tsx`
- **UI role**: `label`
- **User flow**: Users see this option in Settings > Appearance when choosing the app Color Scheme. Selecting it applies the operating system light/dark preference and the Windows accent color when KKTerm starts or when the user switches to this scheme.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: `Match` means follow or mirror the operating system appearance setting, not compare two values for equality.
- **Domain notes**: `OS` means the desktop operating system. The option affects app chrome tokens; terminal pane themes remain controlled by terminal appearance settings.
