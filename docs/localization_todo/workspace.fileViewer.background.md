# workspace.fileViewer.background

- **English value**: `Background`
- **Namespace**: `workspace`
- **File/component**: `src/modules/workspace/connections/file-viewer/FileViewerWorkspace.tsx`
- **UI role**: `button`
- **User flow**: The user sees this option inside the PDF and image/SVG viewer hamburger menu. It opens the shared background picker for the current Document Connection.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: Background means the visual backdrop behind a PDF page or image preview, not a process running in the background.
- **Domain notes**: Document backgrounds are Connection-specific and use the same background picker choices as Dashboard/File Explorer.
